# Text Summarizer Web App

This is a web-based tool that uses AI for text summarization. It takes long input text and generates a shorter summary using the **BART model** from **Hugging Face**.

## Features
- Summarize text using AI
- Simple web interface built with **Flask**
- Copy summary to clipboard
- Easy to use
